## Longer Description (for README or About section):

### Bookstore E-Commerce Platform

A comprehensive full-stack e-commerce website for books built with Django framework. This project demonstrates modern web development practices with a complete online bookstore functionality.

**Key Features:**
- User authentication with email verification and OTP login
- Product browsing, search, and categorization
- Shopping cart and wishlist functionality
- Order management system
- Payment integration (Razorpay)
- Admin dashboard for inventory and order management
- Email notifications and newsletters
- Responsive design with Bootstrap

**Tech Stack:**
- Backend: Python, Django
- Database: SQLite/PostgreSQL
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Payment: Razorpay API
- SMS: Twilio API
- Deployment ready

**Project Structure:**
- Modular Django app architecture
- Custom user model
- RESTful URL patterns
- Template-based rendering
- Static file management

Perfect for learning full-stack development or as a foundation for an online bookstore business.
